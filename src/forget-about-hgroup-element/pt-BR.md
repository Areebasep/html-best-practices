# Esqueça o elemento `hgroup`

Esse elemento não é muito utilizado.

Ruim:

    <hgroup>
      <h1>HTML Best Practices</h1>
      <h2>For writing maintainable and scalable HTML documents.</h2>
    </hgroup>

Bom:

    <h1>HTML Best Practices</h1>
    <p>For writing maintainable and scalable HTML documents.</p>
